# v0.2.0A — CLI Polish

Sub-release of v0.2.0. No version bump.

---

## Fixes

### Intro order
`◇  Registry loaded` was printed before `┌  StackShift Skills Installation` because the registry spinner ran before `intro()`. Fixed by calling `intro()` before the spinner for interactive mode.

### Seed step skipped when user declined tier replacement
Answering "No" to the "Replace with a different tier?" prompt called `cancel()` + `process.exit(0)`, abandoning the session. The seed, scope, and platform prompts were never shown. Fixed: the flow continues with `keepProtocol: true`, skipping only the tier selection block while showing all remaining prompts.

### `keepProtocol` behaviour in writer
When `keepProtocol` is true, `writeSelection` now skips bundle cleanup and re-copy (preserving the existing protocol folder). `writeStackshiftMarker` preserves the existing `mode` and `protocols` fields — only `seed`, `skillVersion`, and `installedAt` are updated.

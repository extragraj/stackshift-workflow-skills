# v0.1.9 — Codex CLI Compatibility

## What changed

Added Codex CLI as a first-class supported platform alongside Claude Code and General agents. All changes are additive — existing Claude Code and General agent installations are unchanged.

---

## CLI changes

### New platform: `codex`

`--platform codex` is now a valid flag value. It installs skills to `.codex/skills/` and generates `AGENTS.md` at the project root (the mechanism Codex CLI uses to discover skills).

```bash
npx @extragraj/stackshift-skills init --platform codex --no-interactive
npx @extragraj/stackshift-skills init --platform claude,codex --no-interactive
```

### AGENTS.md generation (`writer.ts`)

When `codex` is in the selected platforms, the CLI writes or updates `AGENTS.md` with a `<!-- stackshift-skill -->` block. The write is idempotent:

- File absent → created with StackShift section.
- File present, StackShift block found → block replaced in place (safe re-install).
- File present, no StackShift block → appended without touching existing content.

Install output includes `AGENTS.md ← StackShift skill entry written` when applicable.
